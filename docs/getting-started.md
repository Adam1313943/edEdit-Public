# 快速開始 · Getting Started

歡迎使用 edEdit！本頁帶你 5 分鐘上手。

Welcome to edEdit. This page gets you up and running in 5 minutes.

---

## 五個 tab · Five Tabs

iOS 版底部有 5 個 tab：

The iOS app has 5 bottom tabs:

| Tab | 用途 / Purpose |
|---|---|
| 📝 **筆記** Notes | App 內建筆記庫，階層式標籤、iCloud 同步 |
| 📂 **檔案** Files | 從 iCloud Drive、本機、第三方雲端開啟並編輯文字檔 |
| 📑 **比較** Compare | 兩個文字檔案的差異視覺化（Pro） |
| 🔍 **Find** | 在資料夾內遞迴搜尋（Pro） |
| ⚙ **設定** Settings | 購買狀態、版本資訊、回報問題 |

---

## 第一個筆記 · Your First Note

1. 進「筆記」tab → 右上角「新增」（鉛筆圖示）
2. 直接打 Markdown：標題用 `#`、清單用 `-`、粗體 `**bold**`
3. 加標籤：底部 textfield 輸入，用 `/` 分階層，例如 `work/meeting/Q1`
4. 拍立得：按到清單行末 Enter 自動續一個項目；空項目 Enter 跳出清單

> 💡 免費版上限 5 筆筆記。需要更多請升級 Pro。

---

## 開啟磁碟上的檔案 · Open a Disk File

1. 進「檔案」tab → 右上角「+」 → **開啟檔案**
2. 從 Files app picker 中挑任意位置（iCloud Drive、On My iPhone、Dropbox、Google Drive…）
3. 選了之後 edEdit 自動偵測編碼（UTF-8 / Big5 / Shift-JIS 等）與語言
4. 編輯完按右上「儲存」即可寫回原位置

---

## 從 Notes 分享筆記過來 · Share from Apple Notes

1. 在 iOS 內建 Notes app 開一筆筆記 → 點分享按鈕
2. 在分享面板找到 **edEdit**（如果沒看到，往下滑找「更多」加進來）
3. 點下去出現 mini-UI：
   - **預設「存為 edEdit 筆記」** — 直接進到 edEdit 的「筆記」tab，格式（粗體、清單、待辦）會自動轉成 Markdown
   - 也可以選 `.md` / 純文字 `.txt` / 單一 HTML 存成檔案
4. 按「下一步」→ 完成

## 設定 iCloud 同步 · Set Up iCloud Sync

如果你裝置上 iCloud Drive 已開啟，**edEdit 啟動時會自動建立 `iCloud Drive/edEdit/` 資料夾**，並把筆記透過 CloudKit 跨裝置同步。

第一次同步可能需要幾秒到幾分鐘。完成後在 Mac、iPad 上開 edEdit 也能看到同樣的筆記。

詳細請看 [iCloud 同步說明](icloud-sync.md)。

If iCloud Drive is enabled on your device, edEdit automatically creates the `iCloud Drive/edEdit/` folder on launch and syncs notes across devices via CloudKit. First sync may take a few seconds to a few minutes. See [iCloud Sync](icloud-sync.md) for details.
